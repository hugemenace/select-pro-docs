# Custom Predictions — Menu Reference

The menus below are provided by Select Pro and can be surfaced as predictions using `wm.call_menu`. Pass the `bl_idname` as the `name` param.

See [Examples](/custom-predictions/examples) for how to add a menu to the prediction list.

---

## Object Mode

| Label | `bl_idname` |
| :--- | :--- |
| Object Select Menu | `SELECT_PRO_MT_blender_obj` |
| Hierarchy | `SELECT_PRO_MT_obj_hierarchy` |
| More/Less | `SELECT_PRO_MT_blender_obj_select_more_less` |
| Select All by Type | `SELECT_PRO_MT_blender_obj_select_all_by_type` |
| Select Grouped | `SELECT_PRO_MT_blender_obj_select_grouped` |
| Select Linked | `SELECT_PRO_MT_blender_obj_select_linked` |
| Select Similar | `SELECT_PRO_MT_obj_select_similar` |
