# Select Constraint Target

Select the target object of a constraint on the active object. Choose the constraint whose target you want to select; if the active object has more than one constraint of that type, every visible target is selected.

## Properties

| Property | Shortcut | Default | Description |
| :--- | :--- | :--- | :--- |
| Constraint Type | N/A | `Copy Rotation Target` | The constraint target to select |
| Only Enabled | N/A | `True` | Only consider constraints that are enabled |
| Extend Selection | N/A | `False` | Extend the current selection instead of replacing it |
