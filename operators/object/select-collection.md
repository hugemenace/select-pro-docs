# Select Collection

Select all objects belonging to the same collection as the active object. Hold `Alt` to focus the collection in the Outliner instead of selecting its objects.

## Properties

| Property | Shortcut | Default | Description |
| :--- | :--- | :--- | :--- |
| Select Collection Root | N/A | `False` | Select the collection in the Outliner and scroll it into view, instead of selecting its objects |

## Execution Modes

Hold down these keys when executing the operator to change its default behaviour.

| Key | Description |
| :--- | :--- |
| ALT | Sets **Select Collection Root** to `True` |
