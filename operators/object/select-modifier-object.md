# Select Modifier Object

Select the object referenced by a modifier on the active object. Choose the modifier whose object you want to select; if the active object has more than one modifier of that type, every visible object is selected.

## Properties

| Property | Shortcut | Default | Description |
| :--- | :--- | :--- | :--- |
| Modifier Type | N/A | `Boolean Object` | The modifier object to select |
| Extend Selection | N/A | `False` | Extend the current selection instead of replacing it |
